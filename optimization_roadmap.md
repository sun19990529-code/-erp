## 暂缓项（复杂度过高，建议 v2.0+ 独立规划）

| 功能 | 原因 |
|---|---|
| 半成品中间入库 | 需新增工序间暂存仓概念，涉及仓库类型扩展 |
| 工时统计/打卡计时 | 需引入打卡/计时器组件，属独立模块 |
| 设备台账/工位管理 | 需新建 equipment 表，属固定资产范畴 |
| 微信小程序 / PWA | 可基于现有 API 低成本开发，但非紧急 |
| 可视化单据模板编辑器 | 拖拽式编辑器开发量大，先做固定模板 |
