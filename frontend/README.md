# 铭晟管理系统 (ERP-MES)

> **当前版本：v1.4.1** — 安全加固修订版  
> **技术栈**：Node.js + Express + better-sqlite3 | React 19 + Vite 8 + Tailwind CSS

## 项目简介

铭晟管理系统是一套面向中小制造企业的集成 ERP+MES 系统，覆盖从销售、采购、仓储到生产制造的全链路数据协同。

## 核心功能

| 模块 | 功能 |
|------|------|
| 🏭 **生产管理** | 生产工单、APS 甘特图排程、多工序流转、报工大厅 |
| 📦 **仓库管理** | 批次追踪、入库/出库调度、FIFO 库存扣减 |
| 📋 **订单中心** | 销售订单、物料需求、生产联动、出货跟踪 |
| 🔍 **质量检验** | 入库/巡检/委外/成品四类检验 |
| 🛒 **采购管理** | 采购单、收货自动入库 |
| 🏗️ **委外加工** | 委外单管理、工序委外与自动推进 |
| 📊 **数据看板** | 实时 Dashboard、车间大屏看板 |
| 👤 **权限体系** | RBAC 角色权限、API 级别中间件保护 |

## 快速启动

```bash
# 1. 安装依赖（仅首次）
npm run install:all

# 2. 构建前端
cd frontend && npm run build

# 3. 启动系统
node backend/server.js
```

访问 `http://localhost:3198`，默认账号 `admin` / `admin123`。
