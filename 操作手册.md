## 1. 系统概述

### 1.1 系统简介

铭晟管理系统是一套集成ERP与MES功能的企业管理系统，主要功能包括：

- **仓库管理**：原材料、半成品、成品的出入库及库存管理
- **订单管理**：销售订单的录入、跟踪与完成
- **生产管理**：生产工单、工序流转、领料管理
- **质量检验**：入库检验、巡检、委外检验、成品检验
- **采购管理**：采购单据的录入与跟踪
- **委外加工**：外协加工单据管理

### 1.2 系统访问

- **访问地址**：http://localhost:3198
- **默认管理员账号**：请联系系统管理员获取

> **安全说明**：v1.0.2 起，系统采用 bcrypt 加密存储密码。首次登录时旧密码会自动升级为加密存储，无需手动操作。

## 2. 登录与权限

### 2.1 用户登录

1. 打开浏览器，访问系统地址
2. 输入用户名和密码
3. 点击【登录】按钮

### 2.2 权限说明

系统采用角色权限控制，不同角色可访问的功能模块不同：

| 角色类型 | 权限范围 |
|---------|---------|
| 系统管理员 | 全部功能，包括用户管理、权限配置 |
| 生产管理员 | 生产管理、质量检验 |
| 仓库管理员 | 仓库管理、库存查询 |

## 3. 大盘与全景特性(v1.2)
首页登录后即为系统大盘看板，您可以在此处纵深掌控全局业务流：
- **安全库存动态预警**：当任何物资库存量跌破您设置的底水位时，系统除基础发黄灯外，还在看板底部生成了红色血条预警区。（原 `v1.0` 无此功能）。
- **产能宏观时序走势**：通过中部双色连绵的波浪图表（近 7 日），主管可清晰看到进单量与下发加工流的匹配比率。

## 4. 基础数据管理

### 3.1 产品管理

#### 3.1.1 原材料管理

**操作路径**：基础数据 → 原材料管理

**功能说明**：管理原材料的基础信息，包括编码、名称、规格、单位等。

**操作步骤**：

| 操作 | 步骤 |
|-----|------|
| 新增 | 1. 点击【新增】按钮<br>2. 填写产品编码、名称、规格、单位<br>3. 选择类别为"原材料"<br>4. 设置单价、库存预警阈值<br>5. 点击【保存】 |
| 编辑 | 1. 在列表中找到目标产品<br>2. 点击操作列的【编辑】按钮<br>3. 修改信息后点击【保存】 |
| 删除 | 1. 点击操作列的【删除】按钮<br>2. 确认删除操作 |

**字段说明**：

| 字段 | 说明 | 是否必填 |
|-----|------|---------|
| 产品编码 | 唯一标识 | 是 |
| 产品名称 | 产品名称 | 是 |
| 规格 | 产品规格型号 | 否 |
| 单位 | 计量单位（kg、件、支等） | 是 |
| 类别 | 原材料 | 是 |
| 单价 | 单位价格 | 否 |
| 库存预警 | 最低库存阈值 | 否 |

#### 3.1.2 半成品管理

**操作路径**：基础数据 → 半成品管理

操作方式与原材料管理相同，类别选择"半成品"。

#### 3.1.3 成品管理

**操作路径**：基础数据 → 成品管理

**特别说明**：成品新增时需要配置工序流程。

**新增成品步骤**：

1. 点击【新增】按钮
2. 填写基本信息（编码、名称、规格、单位等）
3. 选择类别为"成品"
4. **配置工序流程**：
   - 点击【添加工序】
   - 选择工序类型（轧机、校直、抛光等）
   - 设置工序顺序
   - 勾选是否委外
   - 对于顺序为1的工序，可配置领料清单
5. 点击【保存】

**工序配置说明**：

| 工序 | 编码 | 说明 |
|-----|------|------|
| 轧机 | ROLLING | 轧机加工，可配置领料 |
| 校直 | STRAIGHTENING | 校直处理 |
| 抛光 | POLISHING | 表面抛光 |
| 矫直 | CORRECTING | 矫直处理 |
| 切割 | CUTTING | 切割加工 |
| 拉拔 | DRAWING | 拉拔加工 |
| 清洗 | CLEANING | 清洗处理 |
| 线切割 | WIRE_CUTTING | 线切割加工 |
| 激光切割 | LASER_CUTTING | 激光切割 |
| 热处理 | HEAT_TREATMENT | 热处理加工 |

### 3.2 供应商管理

**操作路径**：基础数据 → 供应商管理

**操作步骤**：

| 操作 | 步骤 |
|-----|------|
| 新增 | 1. 点击【新增】<br>2. 填写供应商编码、名称<br>3. 填写联系人、电话、邮箱、地址<br>4. 点击【保存】 |
| 编辑 | 找到供应商 → 点击【编辑】→ 修改 → 保存 |
| 删除 | 点击【删除】→ 确认 |

### 3.3 客户管理

**操作路径**：基础数据 → 客户管理

**字段说明**：

| 字段 | 说明 |
|-----|------|
| 客户编码 | 唯一标识 |
| 客户名称 | 客户全称 |
| 联系人 | 主要联系人 |
| 电话 | 联系电话 |
| 地址 | 收货地址 |
| 信用等级 | A/B/C等级 |

### 3.4 用户管理

#### 3.4.1 内部用户

**操作路径**：基础数据 → 内部用户

**新增步骤**：

1. 点击【新增】按钮
2. 填写用户名、密码、真实姓名
3. 选择所属部门
4. 选择角色（决定权限范围）
5. 设置状态（启用/禁用）
6. 点击【保存】

#### 3.4.2 外部用户

外部用户适用于供应商或合作伙伴，操作方式与内部用户相同，用户类型选择"外部"。

### 3.5 部门与角色管理

**操作路径**：基础数据 → 部门管理 / 角色管理

- **部门管理**：创建组织架构，为用户分配部门
- **角色管理**：创建角色并分配权限
- **权限管理**：配置各角色的功能权限

### 3.6 材质分类管理（v1.5.0 新增）

**操作路径**：基础数据 → 材质分类管理

**功能说明**：管理材质分类的树形结构（父子级），用于成品与原材料的材质匹配过滤。

**操作步骤**：

| 操作 | 步骤 |
|-----|------|
| 新增父级 | 1. 点击【新增父级分类】<br>2. 填写分类名称<br>3. 点击【保存】 |
| 新增子级 | 1. 在父级分类行点击【添加子级】<br>2. 填写子级名称<br>3. 点击【保存】 |
| 排序调整 | 点击分类行的上移/下移按钮，同层级内排序互换 |
| 删除 | 有子级或被产品引用的分类禁止删除 |

**材质匹配说明**：

- 成品编辑时可选择材质分类
- 在工序物料配置中，原材料下拉框会**实时过滤**为与成品材质相同的原材料
- 无需先保存即可生效，选择材质后立即过滤

## 5. 扫码工作台(v1.2新)
为迎合一线工人的快速操作，系统现已全面接入外部 USB 扫码枪设备。
- 导航进入 **「扫码工作台」**，您无需动用鼠标。
- 只需要拿着扫描枪对准随件打印的 `QR 码` 扣动扳机。系统利用事件探头即刻拦截您的编码。
- **自动分转引擎**：如果您扫入的是入库单码（如 RK-..），系统瞬间切换注入入库办理业务板；如果您扫入生产流转卡（如 PRD-..），系统飞速空降至对应工序进行报工界面。

## 6. 仓库与批次溯源管理

### 6.1 库存查询 (含批号追踪)

**操作路径**：仓库管理 → 原材料库存 / 半成品库存 / 成品库存

**功能说明**：查看当前库存数量、库存状态以及最核心的**来源批号**。
> **v1.2.0革命性升级：** 系统彻底废除过去所有同种货物糊涂加在一起的设计。如今，一笔货物每一次入库（如 `RK-XXX`）都将有唯一的批次标签。同一种产品现在支持在多条不同的批次线中独立存活。

**操作要点**：

- 支持按产品名称、编码以及特定 **批次号** 搜索。
- 库存低于预警值（红线水位）时系统强制标记高亮警报。

### 4.2 入库管理

#### 4.2.1 原材料入库

**操作路径**：仓库管理 → 原材料入库

**新增入库单步骤**：

1. 点击【新增】按钮
2. 选择仓库（原材料仓库）
3. 选择供应商（可选）
4. 添加入库明细：
   - 点击【添加产品】
   - 选择产品
   - 输入库数量
   - 输入公斤数（如需要）
5. 填写经办人、备注
6. 点击【保存】
7. 提交检验（如有检验要求）

**入库单状态流转**：

```
待检验 → 检验中 → 检验通过 → 已入库
                 → 检验不合格 → 退回
```

#### 4.2.2 半成品入库

**操作路径**：仓库管理 → 半成品入库

操作方式与原材料入库相同，选择半成品仓库。

#### 4.2.3 成品入库

**操作路径**：仓库管理 → 成品入库

**特别说明**：成品入库可由生产完工自动生成。

**手动入库步骤**：
1. 选择成品仓库
2. 添加成品明细
3. 关联生产工单（可选）
4. 保存入库单

### 4.3 出库管理

#### 4.3.1 原材料出库

**操作路径**：仓库管理 → 原材料出库

**出库步骤（精确扣指批次版）**：

1. 点击【新增】
2. 选择仓库与出单类型（生产领料/其他出库）
3. 添加出库明细
4. **【核心步骤：指定批次】** 您可以选择从现有库存中下拉圈定特定的一笔入账批次来扣减。如果你未圈定批次，请勿担心：后台将采用智能引擎（按最老的物资，采取 FIFO 先进先出法自动层级扣取！）。
5. 点击【保存】

#### 4.3.2 成品出库

**操作路径**：仓库管理 → 成品出库

**特别说明**：成品出库可由销售订单完成自动生成。

**手动出库步骤**：
1. 选择成品仓库
2. 添加出库产品明细
3. 关联销售订单（可选）
4. 保存出库单

## 5. 订单管理

### 5.1 订单录入

**操作路径**：订单管理 → 订单录入

**新增订单步骤**：

1. 点击【新增】按钮
2. 填写订单基本信息：
   - 选择客户（或输入客户名称）
   - 填写客户电话、地址
   - 选择交货日期
   - 设置优先级
3. 添加订单明细：
   - 点击【添加产品】
   - 选择产品
   - 输入数量、单价
4. 填写备注
5. 点击【保存】

### 5.2 订单状态

**操作路径**：订单管理 → 订单状态

**状态说明**：

| 状态 | 说明 |
|-----|------|
| 待确认 | 订单刚创建，等待确认 |
| 已确认 | 订单已确认，可安排生产 |
| 生产中 | 已生成生产工单，正在生产 |
| 已完成 | 订单已交付完成 |
| 已取消 | 订单已取消 |

**操作要点**：

- 可手动修改订单状态
- 订单完成时自动生成成品出库单
- 可查看关联的生产工单和出库单

### 5.3 订单查询

**操作路径**：订单管理 → 订单查询

**查询方式**：
- 按订单号搜索
- 按客户名称搜索
- 按状态筛选
- 按日期范围筛选

### 5.4 生成生产工单

**操作步骤**：

1. 在订单详情页面，点击【生成生产工单】
2. 系统自动为订单中的每个产品创建生产工单
3. 已存在的生产工单会自动跳过
4. 创建成功后可在订单详情中查看关联的生产工单

## 8. 生产与APS排程(v1.2新)

### 8.1 APS 生产排程面板 (重大革新)
传统的单据堆叠极度抽象，现在您可以直接进入「APS 生产排程面板」以“上帝视角”推演您的产线！
- **流时间线调度**：看板呈左右分页粘连——左侧是尚未终结的生产工单号与物料键；右面是一望无垠的日历刻度轴。系统自动抓取了你的“期望开工”和“期望收尾”生成彩色时间带。
- **色彩辨识**：灰带代表挂起未启动，渐变碧蓝色代指车间正如火如荼进行的战斗单。
- **快接映射**：在甘特板上随意点选任意一条流带，即刻弹回该单身前并允许即时修改它的状态。

### 8.2 生产工单管理

**操作路径**：生产管理 → 生产工单管理

**新增工单步骤**：

1. 点击【新增】按钮
2. 选择产品（仅限成品）
3. 【可选】指派「计划开始日期」与「计划结束日期」（强烈建议设置，它将在 APS 排程控制台具象化生效！）。
4. 输入生产数量、办理流转人等元素。
5. 点击【保存】
> **v1.2新增功能**：单据支持一键【生成并唤醒二维码标识打印】。您可以贴在实物托盘上全厂流转跟踪。

**从订单创建工单**：

1. 在订单详情页面点击【生成生产工单】
2. 系统自动创建关联的生产工单
3. 已存在的生产工单会自动跳过

**工单状态流转**：

```
待生产 → 生产中 → 已完成
```

**完成生产**：

1. 找到目标工单
2. 点击【完成】按钮
3. 系统自动生成成品入库单
4. 库存自动增加

### 6.2 领料管理

**操作路径**：生产管理 → 领料管理

**领料步骤**：

1. 点击【新增】
2. 选择生产工单
3. 选择领料仓库
4. 添加领料明细
5. 保存领料单

**自动领料**：

当成品配置了工序领料清单，在生产工序流转时自动生成领料单。

### 6.3 工序管理

#### 6.3.1 工序流转

**操作路径**：生产管理 → 对应工序（如轧机、校直等）

**操作步骤**：

1. 选择待处理的生产工单
2. 录入本工序的生产数据
3. 选择是否委外
4. 点击【完成本工序】
5. 流转至下一工序

#### 6.3.2 委外工序

当工序勾选"委外"时：

1. 系统自动创建委外加工单
2. 工序状态变为"委外中"
3. 委外完成后，需要进行委外检验
4. 检验通过后工序流转至下一工序

### 6.4 成品工序配置

**操作路径**：生产管理 → 成品工序配置

**功能说明**：为成品配置标准工序流程和物料绑定。

**配置步骤**：

1. 在成品列表中找到目标产品，点击【配置工序】
2. 在弹窗中添加工序：
   - 选择工序类型（轧机、校直、抛光等 10 种）
   - 设置工序顺序
   - 勾选是否委外
3. **配置工序物料绑定（v1.4.0 新增）**：
   - 点击工序行右侧的【物料】按钮，展开物料子面板
   - 点击【添加物料】，选择原材料或半成品
   - 填入**单位用量**（生产1件成品所需的原材料量）和计量单位
   - 已绑定物料的工序，按钮显示物料数量标记
4. 保存配置

> **v1.4.0 核心变更**：首道工序（如轧机）绑定原材料后，报工时系统会**自动从原材料仓库按 FIFO 扣减库存**。如果库存不足，将阻止报工并提示缺料。

## 7. 质量检验

### 7.1 入库检验

**操作路径**：质量检验 → 入库检验

**检验步骤**：

1. 选择待检验的入库单
2. 录入检验数据
3. 填写检验结果：

## 10. 系统联动说明

### 10.1 订单-生产联动

```
销售订单 → 生成生产工单 → 生产完成 → 自动入库 → 订单完成 → 自动出库 → 订单标记已发货
```

**操作流程**：

1. 创建销售订单
2. 在订单详情点击【生成生产工单】
3. 生产工单开始生产
4. 生产完成后自动生成入库单
5. 订单标记完成时自动生成出库单
6. 成品出库完成后，订单自动变更为"已发货"（v1.4.0 新增）

> **v1.0.2**：删除待处理订单时，关联的待处理生产工单会自动清理。有进行中工单的订单禁止删除。
> **v1.4.0**：删除订单时同时清理关联的待处理领料单，杜绝孤儿数据。

### 10.2 生产-仓库联动

```
首道工序报工 → 自动消耗原材料库存（v1.4.0 新增）
生产领料 → 原材料出库
生产完成 → 成品入库（含完工数量同步）
```

> **v1.4.0 核心功能**：首道工序（如轧机）报工时，系统自动按工序绑定的物料清单，乘以产出数量计算实际消耗量，并从原材料仓库按 FIFO 批次扣减。库存不足时阻止报工。
>
> **v1.0.2**：领料完成后自动检查原材料是否全部领齐，全部领齐则标记工单"材料就绪"。

### 10.3 生产-委外联动

```
工序勾选委外 → 自动创建委外单 → 委外完成 → 自动推进工序 → 下一工序或生产完成
```

> **v1.0.2 关键改进**：委外完成时自动推进生产工序，无需手动操作。如为最后一道工序，自动触发完整的生产完成链（成品入库→订单进度更新→成品出库）。

### 10.4 采购-入库联动

```
采购单完成 → 自动创建入库单（含批次号、关联采购单ID）→ 入库检验 → 库存增加
```

> **v1.4.0**：采购自动生成的入库单现在携带完整的批次号（`IN-XXX-1, IN-XXX-2`），不再遗漏。
> **v1.0.2**：入库单关联采购单ID，可从入库记录追溯到源采购单。

### 10.5 采购建议（v1.0.2 新增）

系统根据未完成订单的材料需求缺口，结合当前库存，自动计算需采购量。

**查看方式**：仪表盘 → 采购建议

### 10.6 防重入库机制（v1.0.2 新增，v1.4.0 修复）

| 场景 | 策略 |
|------|------|
| 入库检验通过 | 只更新入库单状态，不直接改库存 |
| 入库单审批 | 统一增加库存，已处理的不重复增加 |
| 成品检验通过 | 只做通过/不通过记录，不修改完工数量（v1.4.0 修正） |
| 委外检验通过 | 只更新委外单状态，库存由委外完成回调统一处理（v1.4.0 修正） |
| 生产完成 | 由报工完成链统一设定完工数量并创建成品入库单 |

### 10.7 质检暂停（v1.0.2 新增）

巡检不合格时，生产工单自动标记为"质检暂停"，仪表盘显示告警数。

### 10.8 关联记录跳转

- **订单详情**：显示关联的生产工单、出库单（可点击跳转）
- **生产工单详情**：显示关联的委外单、入库单（可点击跳转）

## 11. 常见问题

### Q1: 忘记密码怎么办？

请联系系统管理员重置密码。v1.0.2 起密码使用 bcrypt 加密存储，管理员需为用户设置新密码。

### Q2: 为什么无法删除某些数据？

- 已被其他单据引用的数据无法直接删除（级联保护）
- 有进行中生产工单的订单禁止删除
- 系统管理员可使用强制删除功能（会同时删除关联数据）

### Q3: 库存数量不正确怎么办？

1. 检查入库、出库记录是否正确
2. 确认所有入库单是否已通过检验并审批
3. v1.0.2 已修复双重入库问题，如仍有异常请联系管理员

### Q4: 生产工单无法完成？

请确认：
- 所有工序是否已完成流转
- 委外工序是否已检验通过（v1.0.2 起委外完成自动推进工序）
- 成品检验是否已完成
- 工单是否处于"质检暂停"状态（巡检不合格会触发）

### Q5: 生产工单显示"质检暂停"？

巡检不合格时系统自动暂停生产。处理方式：
1. 查看巡检记录，了解不合格原因
2. 处理质量问题后，手动将工单状态改回"生产中"

### Q6: 如何判断是否可以开始生产？

查看生产工单的"材料就绪"标记。当该订单所需的全部原材料领料完成后，系统自动标记为就绪。

### Q7: 如何查看操作历史？

当前版本暂不支持操作日志查询，后续版本会添加此功能。
